# Módulo 3 - Visualização de Dados

Esse é o módulo de Visualização de Dados do Curso de Python para Ciência de Dados. Aqui você aprenderá a transformar dados em gráficos para facilitar a análise e a interpretação das informações.

Neste módulo você aprenderá sobre:

* Fundamentos de visualização de dados
* Biblioteca Matplotlib
* Biblioteca Seaborn
* Gráfico de barras
* Histograma
* Scatter plot
* Boxplot
* Heatmap
* Gráfico de linha
* Boas práticas de visualização

## Instruções para executar o código

### No Google Colab

Você pode abrir os notebooks diretamente no Google Colab de duas formas:

#### Abrir do GitHub

Acesse:
https://colab.research.google.com/github/gustavonogvi/curso-python-ciencia-de-dados/blob/main/modulo3_visualizacao_dados.ipynb

---

### Na sua máquina

Primeiro, clone o repositório:

```bash
git clone https://github.com/gustavonogvi/curso-python-ciencia-de-dados.git
cd curso-python-ciencia-de-dados
```

Crie um ambiente virtual e ative-o:

```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux e MacOS
.\.venv\Scripts\activate   # Windows
```

Instale as dependências:

```bash
pip install jupyter matplotlib seaborn pandas
```

Inicie um servidor Jupyter:

```bash
jupyter notebook
```

Acesse o servidor em:

```
http://localhost:8888
```
