# Módulo 2 - Manipulação de Dados com Pandas

Esse é o módulo de Manipulação de Dados do Curso de Python para Ciência de Dados. Aqui você aprenderá a trabalhar com dados utilizando a biblioteca Pandas, que é uma das principais ferramentas usadas em Ciência de Dados.

Neste módulo você aprenderá sobre:

* Biblioteca Pandas
* Series e DataFrames
* Leitura de arquivos CSV e Excel
* Inspeção de dados
* Limpeza e tratamento de dados
* Estatística descritiva básica
* Filtros e ordenação de dados

## Instruções para executar o código

### No Google Colab

Você pode abrir os notebooks diretamente no Google Colab de duas formas:

#### Abrir do GitHub

Acesse:
https://colab.research.google.com/github/gustavonogvi/curso-python-ciencia-de-dados/blob/main/modulo2_manipulacao_dados.ipynb

#### Fazer upload local

---

### Na sua máquina

Primeiro, clone o repositório:

```bash
git clone https://github.com/gustavonogvi/curso-python-ciencia-de-dados.git
cd curso-python-ciencia-de-dados
```

Crie um ambiente virtual e ative-o:

```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux e MacOS
.\.venv\Scripts\activate   # Windows
```

Instale as dependências:

```bash
pip install jupyter pandas openpyxl
```

Inicie um servidor Jupyter:

```bash
jupyter notebook
```

Acesse o servidor em:

```
http://localhost:8888
```
